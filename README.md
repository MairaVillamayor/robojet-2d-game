# robojet-2d-game
2D game developed with Godot Engine featuring player movement, enemies, lives system and win/lose screens.
Videojuego 2D desarrollado con Godot Engine como parte de mi formación en desarrollo de videojuegos. Incluye mecánicas de movimiento, enemigos, sistema de vidas y pantallas de Game Over y victoria.

🚀 Características
- Sistema de movimiento del personaje
- Enemigos con IA básica
- Sistema de vidas y Game Over
- Pantalla de victoria
- Sonidos y animaciones

🛠️ Tecnologías
- Godot Engine
- GDScript
